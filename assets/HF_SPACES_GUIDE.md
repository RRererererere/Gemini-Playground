# HuggingFace Spaces Integration

## Что это?

Интеграция HuggingFace Spaces позволяет подключать любые AI модели из HuggingFace как скиллы. Gemini сможет автоматически вызывать их для генерации изображений, обработки аудио, анализа данных и многого другого.

## Как это работает

1. **Парсинг API** - система автоматически читает Gradio API схему из `/info` эндпоинта
2. **Создание скилла** - каждый эндпоинт Space становится отдельным tool для Gemini
3. **Автоматический вызов** - Gemini сам решает когда вызвать Space на основе описания
4. **Обработка файлов** - автоматическая загрузка файлов через `/upload` если нужно
5. **Артефакты** - результаты (изображения, аудио) отображаются в чате

## Примеры популярных Spaces

### Генерация изображений

### Видео

- `ali-vilab/i2vgen-xl` - изображение в видео

## Как добавить Space

### Через UI

1. Открой боковую панель
2. Нажми "🤗 HF Spaces"
3. Вкладка "Добавить"
4. Введи URL: `huggingface.co/spaces/owner/space` или просто `owner/space`
5. (Опционально) Введи HF Token для приватных Spaces
6. Нажми "Загрузить Space"
7. Проверь эндпоинты и нажми "Добавить Space"

### Форматы URL

Все эти форматы работают:
- `https://huggingface.co/spaces/black-forest-labs/FLUX.1-schnell`
- `black-forest-labs/FLUX.1-schnell`
- `https://black-forest-labs-flux-1-schnell.hf.space`

## Использование

После добавления Space, Gemini автоматически получает доступ к его функциям:

```
Пользователь: "Сгенерируй картинку с котом в космосе"

Gemini: [вызывает hf_flux1_schnell_0 с prompt="cat in space"]

Space: [генерирует изображение]

Результат: Изображение отображается в чате, Gemini может его описать
```

### С файлами

Если Space требует файл на входе:

```
Пользователь: [прикрепляет фото] "Опиши что на этой картинке"

Gemini: [вызывает hf_blip_0 с прикреплённым файлом]

Space: [анализирует изображение]

Результат: "На изображении изображён закат над океаном..."
```

## Управление Spaces

### Включить/Выключить
Вкладка "Управление" → кнопка питания рядом со Space

### Удалить
Вкладка "Управление" → кнопка корзины

### Обновить
Удали и добавь заново - система загрузит актуальную API схему

## Технические детали

### Поддерживаемые версии Gradio
- Gradio 3.x ✅
- Gradio 4.x ✅

### Типы параметров
Автоматически конвертируются в Gemini tool schema:
- `Textbox` → `string`
- `Number/Slider` → `number`
- `Checkbox` → `boolean`
- `Image/Audio/Video/File` → требует прикреплённый файл

### Типы выходов
Автоматически конвертируются в артефакты:
- Изображения → `SkillArtifact` type `image` (отправляется в Gemini)
- Аудио → `SkillArtifact` type `audio`
- Видео → `SkillArtifact` type `video`
- Текст → `SkillArtifact` type `text`

### Хранение
Все добавленные Spaces хранятся в `localStorage` под ключом `hf_spaces`:
```json
{
  "spaceId": "owner/space",
  "title": "Space Title",
  "apiUrl": "https://owner-space.hf.space",
  "enabled": true,
  "addedAt": 1234567890,
  "token": "hf_...",
  "parsed": { /* API schema */ }
}
```

### CORS
Большинство публичных Spaces настроены на разрешение CORS. Если возникают проблемы - Space может быть приватным или иметь ограничения.

### FLUX.1 - генерация изображений

```

```


## Ограничения

1. **Timeout** - вызовы ограничены 60 секундами
2. **Размер файлов** - зависит от Space (обычно до 10MB)
3. **Rate limits** - некоторые Spaces имеют ограничения на количество запросов
4. **Приватные Spaces** - требуют HF Token

## Troubleshooting

### "Не удалось получить API схему"
- Проверь что Space существует и доступен
- Попробуй открыть `https://owner-space.hf.space/info` в браузере
- Для приватных Spaces добавь HF Token

### "Timeout: Space не ответил"
- Space может быть перегружен или "спящим"
- Попробуй ещё раз через минуту
- Некоторые модели требуют больше времени

### "Требуется прикреплённый файл"
- Эндпоинт требует файл на входе
- Прикрепи файл к сообщению перед отправкой

### "CORS error"
- Space может иметь ограничения CORS
- Попробуй другой Space или обратись к владельцу

## Best Practices

1. **Описывай чётко** - чем точнее промпт, тем лучше результат
2. **Проверяй эндпоинты** - перед добавлением посмотри что Space умеет
3. **Используй популярные Spaces** - они обычно быстрее и стабильнее
4. **Отключай неиспользуемые** - чтобы не перегружать список tools для Gemini
5. **Группируй по задачам** - добавляй только те Spaces которые реально нужны
