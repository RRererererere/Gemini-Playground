# 🎬 Video Frame Extractor — Гайд

Извлекай кадры из видео и анализируй их с помощью AI.

## Быстрый старт

1. **Загрузи видео** в чат (перетащи или через скрепку)
2. **Спроси AI** про конкретный момент
3. **AI автоматически** извлечёт нужный кадр
4. **Анализируй** как обычное изображение

## Примеры запросов

### Базовый анализ

```
"Что происходит на 15 секунде?"
"Опиши сцену в начале видео"
"Покажи кадр на 1:30"
```

### Детальный анализ

```
"Что написано на табличке на 0:45?"
→ AI извлечёт кадр + увеличит нужную область

"Сравни начало и конец видео"
→ AI извлечёт 2 кадра и сравнит

"Найди все моменты где появляется логотип"
→ AI извлечёт несколько кадров
```

### Сохранение в память

```
"Сохрани ключевой момент на 2:15 в память"
→ AI извлечёт кадр + сохранит в визуальную память

"Запомни этот продукт из видео"
→ AI найдёт лучший кадр + сохранит с тегами
```

## Как это работает

1. **AI видит видео** — автоматически получает список (video_1, video_2, ...)
2. **Вызывает tool** — `extract_video_frame(video_1, timestamp=15)`
3. **Кадр извлекается** — через Canvas API в браузере
4. **Показывается в чате** — с индикатором источника и времени
5. **Доступен для анализа** — как обычное изображение

## Визуальный индикатор

Извлечённые кадры помечены фиолетовым индикатором:

```
🎬 demo.mp4  ⏱ 0:15.02
```

Это помогает понять откуда кадр и в какой момент.

## Советы

### Точность таймкода

- Указывай время в секундах: `15`, `90.5`, `125`
- Или в формате: `"0:15"`, `"1:30"`, `"2:05"`
- AI поймёт оба варианта

### Множественные кадры

```
"Покажи кадры каждые 30 секунд"
→ AI извлечёт несколько кадров автоматически

"Сравни 3 разных момента"
→ AI выберет ключевые кадры
```

### Качество

- Кадры сохраняются в JPEG (качество 92%)
- Размер обычно 100-500 KB
- Разрешение = оригинальное разрешение видео

## Ограничения

- Работает только с видео в поддерживаемых форматах (MP4, MOV, WebM)
- Размер видео ограничен памятью браузера
- Извлечение занимает ~100-300ms на кадр

## FAQ

**Q: Можно ли извлечь все кадры?**  
A: Технически да, но это займёт много времени и памяти. Лучше извлекать ключевые моменты.

**Q: Кадры сохраняются?**  
A: Да, в текущем чате. Можно скачать через кнопку Download.

**Q: Можно ли редактировать кадр?**  
A: Нет, но можно использовать zoom_region для увеличения нужной области.

**Q: Работает ли с live видео?**  
A: Нет, только с загруженными файлами.

Автоматически: `{video_name}_frame_{time}s.jpg`

Пример: `demo_frame_15.02s.jpg`
