# Gemini Studio

Полнофункциональная альтернатива Google AI Studio, построенная на Next.js и Gemini API.

## 🚀 Возможности

- 🤖 **Динамическая загрузка моделей** — автоматически получает все доступные текстовые модели Gemini через API
- 🌐 **Live Website Builder** — создавай сайты в реальном времени с AI, взаимодействуй с элементами, drag & drop
- 📝 **Системный промпт** — настройка поведения модели через системные инструкции
- 🧠 **Система памяти** — AI запоминает факты о тебе (локальная и глобальная память)
- 🛠️ **Skills система** — расширяемые инструменты (калькулятор, QR коды, таблицы, URL reader, Website Builder, Image Analyser, Video Frame Extractor)
- 🌡️ **Контроль температуры** — от 0 (точные ответы) до 2 (креативные ответы)
- 📊 **Счётчик токенов** — отображение количества токенов в реальном времени с визуальной шкалой
- ✏️ **Редактирование сообщений** — редактирование как пользовательских, так и ответов ассистента
- 🔄 **Регенерация** — повторная генерация любого ответа модели
- ▶️ **Продолжение генерации** — продолжить генерацию с места остановки
- 📎 **Прикрепление файлов** — изображения (PNG, JPG, WebP, GIF), аудио (MP3, WAV, OGG, M4A), видео (MP4, MOV), текст (.txt), JSON
- 🧠 **Режим глубокого мышления (DeepThink)** — двухпроходный анализ с расширенным контекстом
- 🔑 **Множественные API ключи** — round-robin ротация с per-model блокировкой при rate limits
- 💾 **Управление чатами** — сохранение, загрузка, экспорт/импорт чатов
- 🎨 **Красивый интерфейс** — тёмная тема с плавными анимациями и адаптивным дизайном

## 📦 Установка и запуск

### 1. Установите зависимости

```bash
npm install
```

### 2. Получите API ключ Gemini

Перейдите на [aistudio.google.com/apikey](https://aistudio.google.com/apikey) и создайте бесплатный API ключ.

### 3. Запустите сервер разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) и введите ваш API ключ в боковой панели.

## 💡 Использование

1. **Введите API ключ** в левой боковой панели — модели загрузятся автоматически
2. **Выберите модель** из выпадающего списка (новейшие модели Gemini показаны первыми)
3. **Опционально настройте** системный промпт и температуру
4. **Начните общение** — введите сообщение и нажмите Enter или кнопку "Отправить"
5. **Прикрепляйте файлы** нажав на иконку скрепки или перетащив файлы в окно

### Действия с сообщениями (наведите курсор на любое сообщение)

- **Копировать** — скопировать текст сообщения
- **Редактировать** — редактировать сообщение (редактирование пользовательских сообщений запускает регенерацию)
- **Регенерировать** — сгенерировать новый ответ от модели
- **Продолжить** — продолжить генерацию с места остановки
- **Удалить** — удалить сообщение

### Режим глубокого мышления

Включите переключатель "Deep Think" для активации расширенного режима анализа:
- Модель показывает детальный процесс рассуждений
- Более глубокий анализ сложных вопросов
- Пошаговое объяснение логики ответа

### 🌐 Live Website Builder (NEW!)

Создавай сайты в реальном времени с помощью AI:

1. **Включи Website Builder Skill** в Settings → Skills → Website Builder
2. **Попроси AI создать сайт**: "Создай landing page для кофейни"
3. **Сайт появится справа** в Live Preview панели по мере генерации кода
4. **Взаимодействуй с элементами**:
   - Кликай на элементы → информация попадёт в чат
   - Перетаскивай изображения/текст в чат → AI увидит контекст
   - Модифицируй: "Измени цвет этой кнопки на синий"
5. **Экспортируй результат** через кнопку Download

**Возможности**:
- ✨ Live streaming кода (сайт появляется на глазах)
- 🖱️ Bidirectional communication (клик → контекст)
- 🎨 Drag & Drop элементов из preview в чат
- 📱 Split-screen layout (resizable панели)
- 🎭 Темы и компоненты (navbar, footer, hero, dark mode)
- 💾 Экспорт HTML/CSS/JS

📖 **Подробнее**: [Live Website Builder Guide](./assets/LIVE_WEBSITE_BUILDER_GUIDE.md)

### 🎬 Video Frame Extractor (NEW!)

Извлекай кадры из видео для анализа с помощью AI:

1. **Загрузи видео** в чат (перетащи или через скрепку)
2. **Спроси про конкретный момент**: "Что происходит на 15 секунде?"
3. **AI автоматически извлечёт кадр** и покажет в чате
4. **Анализируй как обычное изображение**:
   - Увеличивай детали (zoom_region)
   - Добавляй аннотации (annotate_regions)
   - Сохраняй в визуальную память (save_image_memory)

**Возможности**:
- 🎥 Извлечение кадров по таймкоду (секунды или номер кадра)
- 🔍 Интеграция с Image Analyser для детального анализа
- 💾 Сохранение важных кадров в память
- 📊 Сравнение кадров из разных моментов
- ⚡ Быстрое извлечение (~100-300ms на кадр)

📖 **Подробнее**: [Video Frame Extractor Guide](./assets/VIDEO_FRAME_EXTRACTOR_GUIDE.md)

## 📁 Структура проекта

```
├── app/
│   ├── api/              # API роуты
│   │   ├── chat/         # Основной чат endpoint
│   │   ├── deepthink/    # Режим глубокого мышления
│   │   ├── models/       # Получение списка моделей
│   │   ├── tokens/       # Подсчёт токенов
│   │   └── translate/    # Перевод текста
│   ├── globals.css       # Глобальные стили
│   ├── layout.tsx        # Корневой layout
│   └── page.tsx          # Главная страница
├── components/           # React компоненты
│   ├── ChatInput.tsx     # Поле ввода сообщений
│   ├── ChatMessage.tsx   # Отображение сообщения
│   ├── DeepThinkToggle.tsx # Переключатель режима
│   └── Sidebar.tsx       # Боковая панель настроек
├── lib/                  # Утилиты и хуки
│   ├── apiKeyManager.ts  # Управление API ключами
│   ├── storage.ts        # Работа с localStorage
│   └── useDeepThink.ts   # Хук для режима глубокого мышления
└── types/
    └── index.ts          # TypeScript типы
```

## 📝 Примечания

- API ключ хранится локально в браузере и никуда не отправляется кроме Google API
- Все настройки сохраняются автоматически
- Поддерживаются все актуальные модели Gemini (1.5 Pro, 1.5 Flash, и др.)
- Файлы конвертируются в base64 перед отправкой в API
