# Agent Creator — Быстрый старт

## Что это?

Создавай персонализированных AI агентов прямо в диалоге с Gemini. Каждый агент — уникальная личность с собственным характером и стилем общения.

## Как использовать?

### 1. Создай агента

Просто попроси:
```
"Создай мне агента-психолога"
"Хочу пообщаться с Шерлоком Холмсом"
"Сделай агента-программиста по Python"
```

### 2. Открой чат с агентом

После создания агент появится в левом сайдбаре (секция "Агенты"). Кликни по нему — откроется новый чат.

### 3. Общайся

Агент будет общаться в своём уникальном стиле, помня свою роль и характер.

## Примеры

**Психолог:**
```
"Создай агента-психолога который специализируется на тревожности"
```

**Персонаж:**
```
"Создай агента на основе Тони Старка из Marvel"
```

**Специалист:**
```
"Создай агента-юриста который разбирается в авторском праве"
```

## Управление

- **Обновить:** "Обнови агента [имя] — сделай его более дружелюбным"
- **Удалить:** Кликни на иконку корзины в сайдбаре
- **Список:** "Покажи всех моих агентов"
