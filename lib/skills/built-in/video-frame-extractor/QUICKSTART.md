# Video Frame Extractor — Quick Start

Извлекай кадры из видео за 3 шага.

## 1. Загрузи видео

Перетащи видео файл в чат или нажми на скрепку.

Поддерживаются: MP4, MOV, WebM

## 2. Задай вопрос

```
"Что происходит на 15 секунде?"
"Покажи кадр на 1:30"
"Сравни начало и конец видео"
```

## 3. Готово!

AI автоматически:
- Извлечёт нужный кадр
- Покажет его в чате
- Проанализирует содержимое

## Примеры

### Базовый анализ
```
👤 Что на видео на 0:30?

🤖 [извлекает кадр]
На кадре видно...
```

### Детальный анализ
```
👤 Увеличь лицо человека на 1:15

🤖 [извлекает кадр + zoom_region]
Вот увеличенное изображение...
```

### Сохранение в память
```
👤 Сохрани этот момент с тегом "важно"

🤖 [извлекает кадр + save_image_memory]
Сохранил в визуальную память!
```

## Советы

✅ Указывай время в секундах: `15`, `90.5`, `125`  
✅ Или в формате: `"0:15"`, `"1:30"`, `"2:05"`  
✅ AI поймёт оба варианта

✅ Можно извлекать несколько кадров:
```
"Покажи кадры на 0:10, 0:30 и 1:00"
```

✅ Комбинируй с другими скиллами:
```
"Извлеки кадр на 0:45 и увеличь логотип"
"Сохрани все важные моменты в память"
```
