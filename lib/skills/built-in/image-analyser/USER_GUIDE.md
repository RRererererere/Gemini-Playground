# 🔍 Image Analyser — Руководство пользователя

## Что это?

Image Analyser позволяет Gemini увеличивать конкретные области изображений для детального анализа. Это полезно когда нужно прочитать мелкий текст, рассмотреть детали или проанализировать конкретную часть изображения.

## Как использовать

### 1. Загрузите изображение

Прикрепите изображение к сообщению через кнопку 📎 или перетащите файл в чат.

### 2. Задайте вопрос о деталях

Gemini автоматически вызовет zoom когда это нужно. Примеры:

**Чтение текста:**
```
"Что написано в правом нижнем углу?"
"Прочитай текст на вывеске"
"Какой номер на футболке игрока?"
```

**Анализ деталей:**
```
"Сколько монет на столе?"
"Какой узор на ткани?"
"Что держит человек на заднем плане?"
```

**Множественные области:**
```
"Сравни текст в левом и правом углах"
"Опиши все лица на фотографии"
```

### 3. Результат

Вы увидите:

1. **Индикатор вызова** — показывает какую область увеличивает Gemini
   ```
   🔍 Zoom region [20%,30% → 60%,70%] — Need to read text
   ```

2. **Превью увеличенной области** — можно кликнуть для просмотра в полном размере

3. **Ответ Gemini** — с информацией из увеличенной области

## Несколько изображений

Если вы загрузили несколько изображений, можно указать какое именно:

```
"Что написано на втором изображении?"
"Увеличь центр первой картинки"
```

Gemini автоматически выберет правильное изображение.

## Советы

### ✅ Хорошие вопросы

- Конкретные: "Что написано в углу?" вместо "Что на картинке?"
- С указанием области: "правый верхний угол", "центр", "задний план"
- О деталях: "мелкий текст", "лица", "номера", "узоры"

### ❌ Плохие вопросы

- Слишком общие: "Опиши изображение" (не требует zoom)
- Без указания области: "Найди текст" (Gemini не знает где искать)
- О всём изображении: "Что на фото?" (zoom не нужен)

## Ограничения

- Работает только с изображениями (не видео)
- Требует наличия изображения в чате
- Максимальное увеличение: 6x
- Качество зависит от разрешения оригинала

## Отключение скилла

Если вы не хотите чтобы Gemini автоматически увеличивал изображения:

1. Откройте **Skills Market** в сайдбаре
2. Найдите **🔍 Image Analyser**
3. Нажмите **Disable**

Скилл останется установленным, но не будет активным.

## Техническая информация

- **Формат вывода**: JPEG (качество 92%)
- **Увеличение**: 2-6x (по умолчанию 3x)
- **Координаты**: В процентах (0-100) от размера изображения
- **Производительность**: ~50-300ms в зависимости от размера

## Часто задаваемые вопросы

**Q: Можно ли увеличить несколько областей одновременно?**

A: Да, Gemini может вызвать zoom несколько раз подряд для разных областей.

**Q: Сохраняются ли увеличенные изображения?**

A: Да, они доступны как артефакты в сообщении и могут быть скачаны.

**Q: Работает ли это с PDF?**

A: Нет, только с изображениями (JPG, PNG, WebP, GIF). Конвертируйте PDF в изображение сначала.

**Q: Можно ли вручную указать координаты?**

A: Да, через Custom Tools можно вызвать `zoom_region` с точными координатами.

**Q: Влияет ли это на токены?**

A: Да, увеличенное изображение отправляется Gemini и считается как дополнительное изображение.

## Поддержка

Если скилл работает некорректно:

1. Проверьте что изображение загружено в чат
2. Убедитесь что скилл активен (Skills Market)
3. Попробуйте более конкретный вопрос
4. Проверьте консоль браузера (F12) на ошибки
