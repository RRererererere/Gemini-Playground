# 🚀 Быстрый старт: Система памяти

## Включение памяти

1. Открой правый сайдбар (кнопка "Настройки")
2. Найди секцию "Инструменты"
3. Включи переключатель "Память"
4. Готово! Модель начнёт запоминать факты о тебе

## Как это работает

### Автоматическое сохранение
Модель сама решает, что важно запомнить:
```
Ты: "Я работаю с Next.js и TypeScript"
Модель: [сохраняет в память] "Работает с Next.js и TypeScript"
```

В чате появится компактный индикатор: 🧠 "Глобальная память сохранена"

### Два типа памяти

**Глобальная** — для всех чатов
- Твои предпочтения
- Технологический стек
- Стиль общения
- Убеждения

**Локальная** — только для текущего чата
- Детали текущей задачи
- Временный контекст
- Специфика проекта

### Использование памяти

Модель автоматически использует сохранённые факты:
```
Ты: "Помоги с роутингом"
Модель: [видит в памяти "Работает с Next.js"]
       "В Next.js 14 используй App Router..."
```

## Управление памятью

Кнопка "Управление памятью" → открывается модал с двумя видами:

### 📋 Список
- Поиск по тексту
- Фильтр по категориям
- Инлайн-редактирование
- Удаление

### 🕸️ Граф
- Визуализация связей между фактами
- Интерактивный: zoom, drag, click
- Размер узла = важность факта
- Цвет = категория

## Категории воспоминаний

- 🎭 **Identity** — личность, имя, роль
- 💻 **Tech** — технологии, языки, фреймворки
- ✍️ **Style** — стиль общения, предпочтения в ответах
- 📁 **Project** — текущие проекты
- ❤️ **Preference** — предпочтения, вкусы
- 💭 **Belief** — убеждения, принципы
- 📖 **Episode** — конкретные события, эпизоды

## Примеры

### Сохранение предпочтений
```
Ты: "Мне нравятся краткие ответы без воды"
Модель: [сохраняет] "Предпочитает краткие ответы"
       → Категория: style
       → Scope: global
```

### Обновление факта
```
Ты: "Теперь я перешёл на Rust"
Модель: [обновляет] "Работает с Next.js" → "Работает с Rust"
```

### Удаление
```
Ты: "Забудь про мой старый проект"
Модель: [удаляет] воспоминание о проекте
```

## Советы

1. **Будь конкретным** — модель лучше запоминает чёткие факты
2. **Проверяй память** — иногда заходи в "Управление памятью"
3. **Используй граф** — визуализация помогает понять связи
4. **Редактируй факты** — если модель что-то неправильно запомнила

## Отключение

Просто выключи переключатель "Память" в настройках.
Все сохранённые факты останутся, но модель перестанет их использовать.
