# Изменения

## Добавлено

### 1. Инструменты (Tools / Function Calling)

- **Секция "Инструменты"** в настройках (правая панель)
- **Конструктор функций** - создание и редактирование инструментов как в AI Studio
- **Глобальное модальное окно** - конструктор открывается поверх всего интерфейса
- **Поддержка вызова функций** - модель может вызывать созданные функции
- **Интерактивный ответ** - пользователь вводит результат выполнения функции
- **Отображение вызовов** - показывает аргументы и результаты функций

#### Как использовать:
1. Откройте настройки (правая панель)
2. Найдите секцию "Инструменты"
3. Нажмите "Создать инструмент" - откроется модальное окно
4. Заполните:
   - Название функции
   - Описание (что делает функция)
   - Параметры (имя, тип, описание, обязательность)
5. Сохраните
6. Модель сможет вызывать эту функцию в чате
7. Когда модель вызовет функцию - введите результат и нажмите "Отправить результаты"

### 2. Редактор системного промпта для DeepThink

- **Кнопка "Редактировать"** в секции "Система"
- **Глобальное модальное окно** с редактором промпта
- **Кнопка "Сбросить"** - возврат к дефолтному промпту
- **Сохранение в чат** - промпт сохраняется вместе с чатом и восстанавливается при загрузке
- **Глобальное сохранение** - промпт также сохраняется глобально для новых чатов

#### Как использовать:
1. Откройте настройки (правая панель)
2. Найдите секцию "Система"
3. Внизу секции найдите "DeepThink промпт"
4. Нажмите "Редактировать" - откроется модальное окно
5. Измените промпт по своему усмотрению
6. Нажмите "Сохранить" или "Сбросить" для возврата к дефолту
7. Промпт сохранится в текущий чат и будет использоваться при его загрузке

### 3. Сохранение системных промптов

- **Кнопка "Сохранить промпт"** под текстовым полем системного промпта
- **Глобальное модальное окно** для ввода названия промпта
- **Список сохранённых промптов** в выпадающем меню

#### Как использовать:
1. Введите системный промпт в текстовое поле
2. Нажмите "Сохранить промпт" - откроется модальное окно
3. Введите название промпта
4. Нажмите "Сохранить"
5. Промпт появится в списке сохранённых

## Архитектура

### Модальные окна
Все модальные окна вынесены на уровень `app/page.tsx`:
- **ToolBuilderModal** - конструктор функций
- **SavePromptDialog** - сохранение системного промпта
- **DeepThinkDialog** - редактор DeepThink промпта

Это обеспечивает:
- Единообразный z-index и overlay
- Централизованное управление состоянием
- Отсутствие вложенных модальных окон

### Состояние
- `showToolBuilder` - показать конструктор функций
- `editingTool` - редактируемая функция (null для новой)
- `showSavePromptDialog` - показать диалог сохранения промпта
- `newPromptName` - название нового промпта
- `savedPrompts` - список сохранённых промптов
- `showDeepThinkDialog` - показать редактор DeepThink
- `deepThinkDraft` - черновик DeepThink промпта

## Технические детали

### Файлы изменены:
- `components/Sidebar.tsx` - удалены модальные окна, добавлены колбэки для открытия
- `components/ChatMessage.tsx` - добавлен компонент `ToolCallsBlock` для отображения вызовов функций
- `app/page.tsx` - добавлены все модальные окна, обработка `functionCall` в стриминге
- `app/api/chat/route.ts` - уже поддерживал вызов функций через Gemini API

### Типы:
- `ChatTool` - описание инструмента
- `ToolCall` - вызов функции моделью
- `ToolResponse` - ответ пользователя на вызов функции
- `ToolSchemaField` - параметр функции
- `SavedChat.deepThinkSystemPrompt` - сохранённый DeepThink промпт в чате
- `SavedSystemPrompt` - сохранённый системный промпт

### API:
- Функции передаются в Gemini API через `tools` параметр
- Модель возвращает `functionCall` в стриме
- Пользователь отправляет `functionResponse` обратно
- Модель продолжает генерацию с учётом результата

### Сохранение:
- DeepThink промпт сохраняется в `localStorage` глобально
- DeepThink промпт сохраняется в каждый чат отдельно
- При загрузке чата восстанавливается его DeepThink промпт
- При создании нового чата используется глобальный промпт
- Системные промпты сохраняются в `localStorage` глобально
