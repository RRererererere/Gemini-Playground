# 🚀 Быстрый тест GeminiBridge

## Шаг 1: Создайте сайт

Скопируйте и отправьте в чат:

```
Создай форму обратной связи с полями: имя, email, сообщение.
Тип сайта: ai_interactive.
При отправке формы данные должны отправляться в чат через GeminiBridge для анализа.
Используй Tailwind CSS для стилизации.
```

## Шаг 2: Проверьте режим

1. Откройте DevTools (F12) → Console
2. Должно быть: `[GeminiBridge] Initialized successfully`
3. Внизу preview должна быть кнопка "AI App" (⚡) — убедитесь что она активна

## Шаг 3: Отправьте данные

1. Заполните форму на сайте
2. Нажмите кнопку отправки
3. Проверьте:
   - ✅ Кнопка заблокировалась и изменила текст на "Отправлено..."
   - ✅ В консоли: `[GeminiBridge] Sending to AI: form_submission {...}`
   - ✅ В чате появился красивый синий блок "Отправка формы"
   - ✅ AI проанализировал данные и ответил

## Ожидаемый результат

В чате должен появиться блок:

```
┌─────────────────────────────────────────┐
│ 📤 Отправка формы                       │
│ form_submission                         │
├─────────────────────────────────────────┤
│ ДАННЫЕ                                  │
│ {                                       │
│   "name": "...",                        │
│   "email": "...",                       │
│   "message": "..."                      │
│ }                                       │
└─────────────────────────────────────────┘
```

Затем AI должен ответить что-то вроде:
```
Получил данные формы обратной связи от [имя].
Email: [email]
Сообщение: [message]

[Анализ тональности и рекомендации]
```

## Другие примеры для тестирования

### Калькулятор
```
Создай калькулятор ИМТ (рост в см, вес в кг).
Тип: ai_interactive.
После расчета результат отправляется AI для интерпретации.
```

### Опрос
```
Создай опрос "Какой ты кофе?" с 3 вопросами.
Тип: ai_interactive.
После завершения AI анализирует ответы.
```

## Отладка

Если не работает:

1. **Проверьте консоль** — должно быть `[GeminiBridge] Initialized successfully`
2. **Проверьте режим** — должен быть "AI App" (⚡), не "Interact" или "Inspect"
3. **Проверьте тип сайта** — в промпте должно быть `website_type: "ai_interactive"`
4. **Перезагрузите preview** — кнопка обновления в правом верхнем углу preview

## Готово!

Если все работает — поздравляю! GeminiBridge полностью функционален. 🎉
