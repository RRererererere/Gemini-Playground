# Быстрый старт: Кастомизация скилов

## За 2 минуты

### 1. Откройте настройки скилла
```
Маркет скиллов → Выберите скилл → ⚙️ Настройки
```

### 2. Выберите что настроить

**📝 Промпт** — описание и системный промпт
**🔧 Инструменты** — описания каждого tool

### 3. Сохраните
```
Кнопка "Сохранить кастомизацию"
```

## Популярные сценарии

### Заставить AI всегда использовать калькулятор

**Скилл:** Calculator  
**Вкладка:** Инструменты → calculate

```
ОБЯЗАТЕЛЬНО используй для ЛЮБЫХ вычислений, даже 2+2.
НИКОГДА не считай в уме. Вызывай ПЕРЕД числовым ответом.
```

### AI активно предлагает сохранять заметки

**Скилл:** Quick Notes  
**Вкладка:** Промпт → Системный промпт

```
Активно предлагай сохранять важную информацию в заметки.
Спрашивай: "Сохранить это в заметки?" когда обсуждается 
что-то важное (факты, задачи, идеи, ссылки).
```

### Читать только техническую документацию

**Скилл:** URL Reader  
**Вкладка:** Промпт → Системный промпт

```
Используй read_url только для технической документации:
API docs, tutorials, GitHub README, technical specs.
Всегда выделяй примеры кода и технические детали.
```

## Экспорт/Импорт

### Экспорт настроек
```
Маркет скиллов → 📥 Download → Сохранить JSON
```

### Импорт настроек
```
Маркет скиллов → 📤 Upload → Выбрать JSON
```

**Применение:**
- Перенос между устройствами
- Бэкап конфигурации
- Шаринг с командой

## Сброс к умолчаниям

```
Настройки скилла → Любая вкладка → "Сбросить к умолчаниям"
```
