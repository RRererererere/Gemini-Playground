# File Editor — Quick Start

## 🚀 Быстрый старт за 3 шага

### 1️⃣ Прикрепите файл

Перетащите `.ts`, `.js`, `.py`, `.html` или любой другой code-файл в чат.

### 2️⃣ Попросите AI изменить код

```
Добавь обработку ошибок в функцию fetchData
```

### 3️⃣ Просмотрите и примените изменения

- 🟢 Зелёные строки = добавлено
- 🔴 Красные строки = удалено
- ✅ Accept = применить
- ❌ Reject = отменить

## 🎨 Режимы просмотра

- **Diff** — построчное сравнение с цветами
- **Editor** — редактирование вручную
- **Split** — оригинал vs изменённый side-by-side

## 💡 Советы

1. AI автоматически использует diff-based подход (не переписывает весь файл)
2. Можно открыть несколько файлов — переключайтесь через табы
3. История изменений доступна через dropdown "История"
4. Кнопка Revert вернёт к оригиналу
5. Download сохранит файл на диск

## 🔗 Интеграция с Website Builder

Если вы создали сайт через Website Builder:

```
Покажи код сайта
```

AI экспортирует HTML в File Editor для прямого редактирования.

## ⚙️ Технические детали

- Поддержка 20+ языков
- Fuzzy matching для надёжного поиска
- Лимит: 100KB текста на файл
- Хранение: per-chat (не глобально)
