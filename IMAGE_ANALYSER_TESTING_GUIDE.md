# Image Analyser — Testing Guide

## Как протестировать новые функции

### 1. Новый UI (ImageLightbox)

**Тест 1: Открытие изображения из чата**
1. Загрузи изображение в чат (drag-and-drop или кнопка прикрепления)
2. Кликни на изображение
3. ✅ Должен открыться lightbox с backdrop blur
4. ✅ Вокруг изображения должна быть рамка с "негативным свечением"
5. ✅ В топбаре должен быть ID изображения (ph_xxxxxxxx)
6. ✅ Внизу должна быть инфобар с метаданными

**Тест 2: Зум и pan**
1. Открой изображение в lightbox
2. Покрути колесо мыши
3. ✅ Изображение должно зумиться плавно
4. ✅ При зуме > 1 курсор должен стать "grab"
5. Зажми и потяни изображение
6. ✅ Изображение должно двигаться (pan)
7. Двойной клик
8. ✅ Зум должен сброситься к 1.0

**Тест 3: ID badge в чате**
1. Наведи на изображение в чате
2. ✅ В левом верхнем углу должен появиться badge с ID (ph_xxxxxxxx)

### 2. AI-аннотации

**Тест 1: Создание аннотаций**
1. Загрузи изображение
2. Попроси нейронку: "Покажи где находится кнопка на этом скриншоте"
3. ✅ Нейронка должна вызвать `annotate_regions`
4. ✅ Должен появиться артефакт с аннотированным изображением
5. ✅ На изображении должна быть синяя рамка с лейблом и стрелкой

**Тест 2: Разные типы аннотаций**
Попроси нейронку создать аннотации разных типов:
- `highlight` — жёлтая пунктирная рамка
- `pointer` — синяя рамка + стрелка
- `warning` — красная пульсирующая рамка
- `success` — зелёная рамка
- `info` — фиолетовая рамка

**Тест 3: Hover эффект**
1. Наведи на аннотацию
2. ✅ Лейбл должен увеличиться
3. ✅ Рамка должна подсветиться

### 3. Уникальные ID

**Тест 1: Генерация ID**
1. Загрузи несколько изображений
2. Открой DevTools → Application → IndexedDB → gemini_studio_files
3. ✅ Все ключи должны иметь формат `ph_xxxxxxxx`
4. ✅ Все ID должны быть уникальными

**Тест 2: Копирование ID**
1. Открой изображение в lightbox
2. Кликни на ID в топбаре
3. ✅ ID должен скопироваться в буфер обмена
4. Вставь (Ctrl+V) в текстовое поле
5. ✅ Должен вставиться ID формата `ph_xxxxxxxx`

### 4. Вставка через Ctrl+V

**Тест 1: Вставка из буфера**
1. Скопируй изображение в буфер обмена (например, из браузера)
2. Кликни в textarea чата
3. Нажми Ctrl+V
4. ✅ Изображение должно добавиться в список прикреплённых файлов
5. ✅ В textarea НЕ должен вставиться base64 текст

**Тест 2: Вставка текста**
1. Скопируй обычный текст
2. Кликни в textarea
3. Нажми Ctrl+V
4. ✅ Текст должен вставиться нормально (не перехватывается)

### 5. URL-доступ

**Тест 1: Прямой доступ по ID**
1. Загрузи изображение в чат
2. Скопируй его ID (например, `ph_a3f9k2m1`)
3. Открой в новой вкладке: `http://localhost:3000/photo/ph_a3f9k2m1`
4. ✅ Должно открыться изображение в lightbox
5. ✅ Должны быть видны метаданные (имя, размер, тип)

**Тест 2: Неверный ID**
1. Открой: `http://localhost:3000/photo/invalid_id`
2. ✅ Должна показаться ошибка "Неверный формат ID изображения"

**Тест 3: Несуществующий ID**
1. Открой: `http://localhost:3000/photo/ph_00000000`
2. ✅ Должна показаться ошибка "Изображение не найдено"

## Проверка совместимости

### Обратная совместимость
1. Открой старый чат с изображениями (если есть)
2. ✅ Все изображения должны отображаться корректно
3. ✅ Старые артефакты должны работать

### Разные типы файлов
1. Загрузи PNG, JPG, WEBP, GIF
2. ✅ Все форматы должны работать
3. Загрузи PDF, аудио, видео
4. ✅ Не должны получать image ID (только изображения)

## Известные ограничения

1. **URL-доступ работает только для текущей сессии**
   - Изображения хранятся в IndexedDB браузера
   - После очистки данных браузера изображения пропадут

2. **Аннотации не сохраняются на исходном изображении**
   - Создаётся новый артефакт с аннотациями
   - Исходное изображение остаётся без изменений

3. **ID badge виден только при hover**
   - Чтобы не загромождать UI
   - Всегда виден в lightbox

## Что делать при ошибках

### Изображение не открывается
1. Проверь консоль браузера (F12)
2. Проверь IndexedDB: DevTools → Application → IndexedDB
3. Убедись что файл там есть

### Аннотации не отображаются
1. Проверь что нейронка вызвала `annotate_regions`
2. Проверь координаты (должны быть 0-100)
3. Проверь что sourceImageId существует

### ID не копируется
1. Проверь что браузер поддерживает Clipboard API
2. Попробуй в HTTPS (некоторые браузеры требуют)

### Ctrl+V не работает
1. Убедись что фокус в textarea
2. Проверь что в буфере именно изображение
3. Проверь консоль на ошибки

Если все тесты прошли успешно:
- ✅ Новый UI работает
- ✅ Аннотации отображаются
- ✅ ID генерируются и отображаются
- ✅ Ctrl+V работает
- ✅ URL-доступ работает
